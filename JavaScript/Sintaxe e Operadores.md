# Sintaxe e Operadores



## ATRIBUIÇÃO



=   -->    x=y  -->   x = y

+=  -->  x +=y  -->  x = x+y

*=  -->  x *= y  -->   x = x * y

/=  --> x /= y  -->  x = x / y

%=  --> x %= y  -->   x % y  (retorna o resto da divisão)



## ARITIMÉTICA

## COMPARAÇÃO
